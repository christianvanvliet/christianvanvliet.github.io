# christianvanvliet.com

A personal research archive for published papers, conjectures, and supporting materials.

This repository contains the source for christianvanvliet.com, a simple publication-first website designed to house long-form work in a durable and readable format.

The archive currently begins with **The Van Vliet Bitcoin Price Conjecture** and will expand over time to include supporting articles, charts, tables, and related essays.

## Site map

- `/` — Home
- `/research/` — Research archive
- `/research/bitcoin-price-conjecture/` — Featured paper
- `/about/` — Purpose of the archive
- `/contact/` — Correspondence
- `/files/` — PDF editions and related documents

## Purpose

The goal of the site is straightforward: to provide a stable public home for the work and the materials that support it.
